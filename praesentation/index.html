Lesson Versiegelte Klassen (sealed class)

https://kotlinlang.org/docs/reference/sealed-classes.html

Versiegelte Klassen werden zur Darstellung eingeschränkter Klassenhierarchien verwendet, wenn ein Wert einen der Typen aus einer begrenzten Menge haben kann, aber keinen anderen Typ haben darf.

sealed class Haustier(open val alter: Int)

class Katze(val fellfarbe: String, alter: Int) : Haustier(alter)
class Kaiman(val gefresseneGnus: Int, alter: Int) : Haustier(alter)

fun `beschreibe Haustier`(haustier: Haustier) = when (haustier) {
    is Katze -> println("Der Fell der Katze ist $haustier.fellfarbe.")
    is Kaiman -> print("Der Kaiman hat ${haustier.gefresseneGnus} Gnus gefressen.")
}
